# Mathematical Modelling of the Mitochondrial Dicarboxylate Carrier (SLC25A10)

This repository contains the core MATLAB codes and supporting files for our revised manuscript:
**“Mathematical Modelling of the Mitochondrial Dicarboxylate Carrier (SLC25A10)”**.

Because the full set of scripts and outputs used to reproduce *every* figure and table in the revised article is large, the complete reproducibility bundle is archived on **Zenodo** (see below). This GitHub repository provides the main codebase and a lightweight structure for navigating the project.

---

## Repository structure (GitHub)

- `ODE_CODES/`  
  Core ODE model implementations and simulation scripts.

- `Bayesian_MCMC_MH_CODES/`  
  MCMC / Bayesian inference scripts (Metropolis–Hastings) and posterior-processing utilities.

- `Experimental_Datasets/`  
  Experimental datasets used in the revised manuscript (or pointers/processed inputs, depending on size).
